# Welcome to Syqlorix

Syqlorix is a hyper-minimalist Python framework designed for developers who love Python. It lets you build beautiful, dynamic web frontends with an intuitive live-reload server and a first-class developer experience—all without the complexity of modern JavaScript.

This documentation will guide you from initial setup to final deployment. Let's get started!

> [!NOTE]
> The best way to begin is by following the **Getting Started** guide after reading this introduction. It will walk you through your first project in minutes.

## Core Philosophy

Our design is guided by three core principles to ensure a productive and enjoyable development process.

1.  **Python-Native:** If you know Python, you know Syqlorix. We believe in leveraging the clarity and power of Python for everything from page structure to server logic.
2.  **Minimalism:** No unnecessary dependencies or complex configurations. Get up and running in seconds with a tiny, easy-to-learn API.
3.  **Developer Experience:** Features like live-reload, automatic static file handling, and a beautifully branded 404 page are built-in to make development a joy.

> [!TIP]
> Use the `syqlorix init` command to instantly create a new project with a feature-rich example file.

> [!WARNING]
> For production deployment, always use the `syqlorix build` command. The `run` command is intended for development use only.
